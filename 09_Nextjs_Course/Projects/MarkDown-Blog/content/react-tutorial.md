## Introduction to React

React is a **popular JavaScript library** developed by **Facebook (Meta)** for building **fast, interactive, and scalable user interfaces**.  
It follows a **component-based architecture**, which allows developers to break the UI into small, reusable pieces.

React is mainly used for building **Single Page Applications (SPAs)** where data updates dynamically without reloading the page.

---

## Why Learn React?

React is one of the most in-demand frontend technologies today. Here’s why you should learn it:

- 🧩 **Component-Based Architecture** – Build reusable and modular UI components
- ⚡ **Fast Performance** – Uses a Virtual DOM for efficient updates
- 📈 **High Industry Demand** – Used by Facebook, Netflix, Airbnb, Instagram, and more
- 🌍 **Strong Ecosystem** – Works well with Next.js, Redux, Tailwind CSS, and many libraries
- 💼 **Career Growth** – Essential skill for Frontend & Full Stack developers

---

## Prerequisites

Before learning React, you should have basic knowledge of:

- HTML & CSS
- JavaScript (ES6 concepts like `let`, `const`, arrow functions)
- Basic understanding of the DOM

If you know **JavaScript well**, learning React becomes much easier 🚀.

## Setting Up React Environment

To start working with React, you need a basic development setup.

### 1. Install Node.js

Download and install **Node.js** from the official website.  
It includes **npm (Node Package Manager)**, which is required to manage React packages.

### 2. Create a React App

The easiest way to start with React is using **Create React App**.

```bash
npx create-react-app my-app
cd my-app
npm start
