# What is Python?

Python is an **open-source programming language** created to make coding **simple and readable**.

In simple words:

> **Python = Easy syntax + Powerful features 💡**

Python supports **multiple programming styles**:
- Procedural
- Object-Oriented
- Functional

# History of Python

- 📅 **1991** – Python was created by **Guido van Rossum**
- 🐍 Named after *Monty Python*
- 🎯 Goal: Simple and readable language
- 📈 Rapid growth in popularity

Today, Python is used by companies like **Google, Netflix, Instagram, Spotify**.
